Popovers

The Popover component is similar to tooltips; it is a pop-up box that appears when the user clicks on an element. The difference is that the popover can contain much more content.

Popover Positioning

By default, the popover will appear on the right side of the element. Use the data-bs-placement attribute to set the position of the popover on top, bottom, left or the right side of the element:
